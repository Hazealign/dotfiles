## Install

```sh
git clone --recursive https://github.com/Hazealign/dotfiles.git ~/.dotfiles
cd ~/.dotfiles;
chmod a+x ./dependencies/{OS}.sh; source ./dependencies/{OS}.sh; # Optional, Put your OS.
npm install; npm start;
```
